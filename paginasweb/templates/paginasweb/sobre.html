Sobre

A disciplina de Programação para Web tem como objetivo apresentar os conceitos fundamentais de desenvolvimento web, abrangendo tanto o front-end quanto o back-end. Os alunos aprenderão a criar páginas web dinâmicas e interativas, utilizando tecnologias como HTML, CSS, JavaScript e frameworks populares. Além disso, serão introduzidos aos princípios de design responsivo e acessibilidade, visando garantir que as aplicações sejam funcionais e agradáveis em diferentes dispositivos e para todos os usuários.
